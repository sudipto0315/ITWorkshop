
# IT Workshop



## Usage

To navigate through this project go through this path

```path
  ~ITWorkshop/labNumber/src/exampleCode.java
```


## Run Locally

Clone the project

```bash
  git clone https://github.com/sudipto0315/ITWorkshop.git
```

Go to the project directory

```bash
  cd ITWorkshop
```
